Who I am

Based out of San Diego

Over the years I have become passionate about the technological and scientific industry. Although being a skillful statistician and a keen programmer are important, other qualities for the workforce are necessary. Honesty, diligence, teamwork, patience, and respect are important attributes one must possess to ensure a proper work environment in order to produce a competent performance. I continue to research and implement tasks related computer hardware and software, programming languages, artificial intelligence, data analysis techniques, in order to properly understand what employers might consider valuable for their company.

I currently build websites using HTML, CSS, Javascript and Django. I use Python libraries like numpy, pandas, seaborn, scikitlearn, matplotlib for analysis and visualizations. Other work includes the use of Keras/Tensorflow for Machine Learning, Deep Learning, and TinyML (embedded systems). All while sharpening my skills in Big Data tools like pySpark and map Reduce, AWS for cloud services, C/C++ for data structures and memory optimization. Lastly, I implement penetration testing for information security using Kali Linux through a virtual machine.
